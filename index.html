About

I got into programming by shipping something real — I built and published a game on Steam while still in high school. That set the trajectory for everything that followed.

Now I'm an MSE student in CSE at the University of Michigan, where I also teach as a GSI for EECS 281 (Data Structures & Algorithms) and EECS 291 (Intro to Applied Data Structures & Algorithms).

These days my focus is on AI systems — what it looks like when agents stop being toys and start doing real work. I'm building a multi-agent orchestration platform that runs 24/7, handling programming, research, writing, and code review autonomously.

This summer I'm interning at Microsoft on Azure hyperscale networking.
